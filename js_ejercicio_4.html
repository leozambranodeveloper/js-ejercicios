4. Filtro de Búsqueda en Tiempo Real

Crea una página con un campo de texto y una lista predefinida de elementos. ● Mientras el usuario escribe en el campo, la lista debe actualizarse en tiempo real para mostrar solo los elementos que contienen el texto escrito. Ejemplo: Si la lista contiene ["Perro", "Gato", "Pez"] y el usuario escribe "Ga", solo "Gato" debe quedar visible.

  • Perro
  • Gato
  • Pez
  • Ave
  • Leon
  • Raton
